# MasjidConnect GY

**The Islamic community app for Guyana** — prayer times, Quran, fasting tracker, masjid directory, comprehensive Islamic education, and community features built for the Guyanese Muslim community.

**Live:** https://masjidconnectgy.com | **GitHub:** https://github.com/kareemschultz/v0-masjid-connect-gy

---

## Features

### Prayer & Worship
- **Prayer Times** — accurate times for Georgetown, Guyana (Muslim World League / Egyptian General Authority)
- **Adhan Countdown** — live countdown to next salah, all 5 prayers displayed
- **Prayer Tracker** — log your 5 daily prayers with streak counter and 7-day chart
- **Sunnah & Nawafil Tracker** — Witr, Fajr Sunnah, Duha, Tahajjud, Ishraq, Awabeen, Tarawih
- **Qibla Compass** — direction to Makkah (iOS permission-aware)
- **Adhkar** — morning and evening supplications with counter

### Quran
- **Mus'haf** — page-by-page Quran (604 pages, Amiri Quran font)
- **Recitation** — audio with 12 reciters (Mishary Alafasy, Sudais, Maher al-Muaiqly and more)
- **Hifz Mode** — memorisation aid with reveal/hide mechanic
- **Continue Reading** — saves last page with progress bar
- **Quran Verse Sharer** — share any ayah via WhatsApp/text/copy

### Ramadan
- **Ramadan Companion** — suhoor/iftaar countdowns, Hijri date, moon sighting selection (CIOG / Saudi/GIT)
- **GIT Official Timetable** — full 30-day 1447 timetable for Georgetown with regional adjustments
- **Fasting Tracker** — year-round: Ramadan, Shawwal (6 days), Mon/Thu, Ayyam al-Bayd (13/14/15), Voluntary
- **Ibadah Tracker** — prayer log, Quran pages, Sadaqah (GYD), Adhkar, Istighfar, Sleep, Water, Good Deeds
- **Factors Affecting the Fast** — what invalidates vs. what doesn't (Darul Uloom reference)
- **Fidya & Missed Fasts** — who is exempt, fidya rulings, Sadaqatul Fitr guide
- **Sisters in Ramadan** — 8 worship actions for menstruating women

### Masjid Directory
- **31 masjids** across Georgetown, East Coast Demerara, East Bank Demerara, Berbice, Linden, West Demerara
- Search, area filter, facility badges (Parking, Women's Section, Jumu'ah, Wudu Area, Islamic Store)
- Masjid detail page: prayer times table, imam info, about section
- **My Masjid** — pin your home masjid, appears first in list
- Proximity sort for remaining masjids
- Masjid check-in with live daily counter
- Jumu'ah Friday pulse banner

### Community
- **Community Dua Board** — share prayer requests, react with Ameen
- **Quran Khatam Collective** — claim a Juz for a community Khatam
- **Personal Khatam Tracker** — 30-Juz personal progress grid
- **Community Feed** — share reminders, questions, announcements
- **Buddy System** — faith partner streaks, 5-level ranking (Seeker → Devoted → Steadfast → Illuminated → Champion)
- Find buddies by @username, phone number, or email
- Live challenge progress: Witr Streak, Tahajjud Week, Duha Month, Nawafil Sprint

### Madrasa (Islamic Education)

#### Quran Learning
- **Noorani Qaida** — 12 lessons, 29 Arabic letters with makhraj (points of articulation)
- Interactive letter forms viewer (initial, medial, final)
- Tajweed rules: Noon Sakinah, Raa, Laam, Waqf

#### Salah Education
- **Learn to Pray** — Wudu (8 steps) + Salah (10 steps) with SVG posture icons
- **Nawafil tab** — Sunnah and optional prayers explained
- **All Islamic Prayers** — complete taxonomy with rakat, timing, hadith rewards

#### Fiqh (Islamic Law — Hanafi)
- **Fiqh Guide** — sourced from Al Fiqh-ul Muyassar (Hanafi):
  - **Taharah (Purification)** — 5 types of water, wudu pillars (4 fard), types of filth, ghusl, tayammum
  - **Salah** — conditions, pillars, sunnahs, what invalidates
  - **Sawm (Fasting)** — what invalidates, expiation, who is exempt
  - **Zakah** — nisab, who must pay, the 8 asnaf
  - **Nikah & Family** — marriage conditions, rights, divorce
  - **Halal & Haram** — food, drink, earnings
  - **Seeking Knowledge** — obligation, etiquette, sources

#### Islamic History & Knowledge
- **Seerah** — Life of the Prophet ﷺ across 4 eras
- **Stories of 25 Prophets** — from Adam (AS) to Isa (AS)
- **Islamic Adab** — 9 etiquette topics (greetings, eating, entering home, entering masjid, sneezing/yawning, speaking, parents, neighbours, **bathroom etiquettes**)
- **Islamic Names** — 80+ names with Arabic, meaning, gender, origin filter
- **Arabic Typing Practice** — keyboard familiarity tool

#### GII Islamic Library
- Scribd iframe reader — read in-app without leaving to browser
- **Aqeedah Tahawiyyah** (151 pages) — Commentary on Islamic creed
- **Practical Essentials for Muslim Students** (40 pages) — Student fiqh guide (GII)
- **Islamic Adab** (GII) — manners and conduct
- **Tafsir Ibn Kathir — Juz' Amma** — Quranic exegesis
- Fiqh-us-Seerah, Islamic Jurisprudence Vol. 1 & 2 — coming (awaiting embed keys from GII)

## Tech Stack

- **Framework:** Next.js 14 (App Router, TypeScript)
- **UI:** Tailwind CSS + shadcn/ui (permanently dark — `bg-[#0a0b14]` base)
- **Auth:** Better Auth with Google Sign-In, @username + phone fields
- **Database:** PostgreSQL (kt-central-db container)
- **Audio:** archive.org (free Islamic audio, no DRM)
- **Quran Reciters:** Quran.com CDN
- **Push Notifications:** Web Push API (VAPID) + `node-cron` server-side scheduler
- **Prayer Times:** adhan.js (Muslim World League, Georgetown 6.8013°N 58.1553°W)
- **Fonts:** Amiri + Amiri Quran (Arabic), Geist (Latin)
- **Deployment:** Docker on kt-nexus-01 via Pangolin reverse proxy

## Deployment

```bash
# Build
docker build -t ghcr.io/kareemschultz/v0-masjid-connect-gy:latest .

# Run (3 networks required)
docker run -d --name kt-masjidconnect-prod --restart always \
  --network pangolin --ip 172.20.0.24 \
  --env-file .env.local \
  ghcr.io/kareemschultz/v0-masjid-connect-gy:latest
docker network connect kt-net-apps kt-masjidconnect-prod
docker network connect kt-net-databases kt-masjidconnect-prod
```

### Environment Variables
```
DATABASE_URL=postgres://mcgy_user:...@kt-central-db:5432/masjidconnect
BETTER_AUTH_SECRET=...
```
